# My Nutrition App

A personalized nutrition and wellness app built with Expo and React Native.

## Features

- **Recipe of the Day** - Browse recipes with ingredients, instructions, and nutritional info
- **Nutrition Plan** - View a personalized nutrition plan
- **Guidelines** - General nutrition guidelines and recommendations
- **Articles** - Read health and nutrition articles
- **Calculator** - Built-in calculator for quick math

## Getting Started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

## Tech Stack

- [Expo](https://expo.dev) with [expo-router](https://docs.expo.dev/router/introduction/) (file-based routing)
- React Native
- TypeScript
- Next.js (web support)
